# Потерянное будущее

Современный одностраничный веб-сайт с тёмной меланхоличной атмосферой, созданный с использованием React, Tailwind CSS и TypeScript.

## Особенности

- **Тёмная синяя палитра** — грустная и глубокая атмосфера
- **Адаптивный дизайн** — отлично выглядит на всех устройствах
- **Секция Hero** — впечатляющее начало с заголовком и описанием
- **Интерактивные карточки биографий** — плавные анимации и эффекты при наведении
- **Квадратные изображения** — строго квадратный формат для всех портретов
- **Современная типография** — читаемый и выразительный текст

## Технологический стек

- **React 19** — современная библиотека для создания UI
- **Tailwind CSS 4** — утилиты для быстрого стилизирования
- **TypeScript** — типобезопасность и лучшая разработка
- **Vite** — быстрый сборщик и dev сервер
- **shadcn/ui** — готовые компоненты UI

## Установка и запуск

### Требования
- Node.js 18+
- npm или pnpm

### Установка зависимостей
```bash
pnpm install
```

### Запуск dev сервера
```bash
pnpm dev
```

Сайт будет доступен по адресу `http://localhost:3000`

### Сборка для production
```bash
pnpm build
```

### Preview production сборки
```bash
pnpm preview
```

## Добавление биографий

Откройте файл `client/src/pages/Home.tsx` и отредактируйте массив `biographies`:

```typescript
const biographies = [
  {
    id: 1,
    name: "Имя человека",
    description: "Описание биографии",
    imageUrl: "https://example.com/image.jpg", // URL изображения
  },
  // Добавьте больше биографий...
];
```

## Кастомизация стилей

Все цвета и переменные стилей находятся в файле `client/src/index.css`. Вы можете изменить:

- **Цветовую палитру** — измените значения CSS переменных в `:root` и `.dark`
- **Шрифты** — обновите `--font-display` и `--font-body` в `client/index.html`
- **Радиус скругления** — измените `--radius` в CSS переменных

## Развёртывание

Сайт готов к развёртыванию на любой платформе, поддерживающей Node.js:

- **Manus** — встроенная поддержка
- **Vercel** — `pnpm build && pnpm start`
- **Netlify** — статическая сборка
- **Railway, Render** — Docker контейнеризация
